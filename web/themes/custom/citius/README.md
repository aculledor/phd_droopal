# Prerequisites

Theme is based on:
 - Single Directory components (SDC)

# Hints for developers

## Components
Components styles and scripts are placed directly in component folder.

### Other styles and scripts
It is not always makes sense to redefine templates with SDC. For example styles of forms, scripts or some elements provided by core or contrib modules are still handled as usual by styles from `src/css` and `src/js` folders.
